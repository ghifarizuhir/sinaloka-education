9. Specs
Elements: 3 stacked books (asymmetric, -3° rotation) + star accent(s)
Colors: Teal gradient (#0f766e → #14b8a6), Amber books (#F59E0B), Yellow stars (#FDE68A), White semi-transparent books
App variants: Same book stack, different background gradient (Teal = Platform, Green = Tutors, Purple = Parent)
Corner radius: 112/512 = 21.875% (Apple standard)
Safe zone (Android adaptive): All elements within 72dp center
Sizes needed: 1024×1024 (App Store), 512×512 (Play Store), 192×192 (PWA), 180×180 (apple-touch-icon), 32×32 (favicon), 16×16 (tiny)
Font: Plus Jakarta Sans 800 for wordmark
